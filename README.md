# Network Simulator

To run the file

```
java -cp ./build/bin/ main.java.net.networkSimulator.model.devices.EndDevice
```

To re-compile the file

```
javac -d ./build/bin/ ./src/main/java/net/networkSimulator/model/devices/Wire.java ./src/main/java/net/networkSimulator/model/devices/Port.java ./src/main/java/net/networkSimulator/model/devices/EndDevice.java ./src/main/java/net/networkSimulator/model/devices/Device.java ./src/main/java/net/networkSimulator/model/devices/Hub.java ./src/main/java/net/networkSimulator/model/devices/Bridge.java
```
